cd <directory> | si posiziona nella directory selezionata

git config | global user.email "<email>" | imposta l'email dell'utente git (obbligatorio per poter usare GIT)

git config | global user.name "<username>" | imposta l'username dell'utente git (obbligatorio per poter usare GIT)

git init | imposta la directory selezionata come repository

git add <file> | aggiunge un file della directory nell'index di git, cioè una "cartella intermedia" dove inserire i file su cui eseguire il commit (eseguendo "git add ." aggiunge tutti i file presenti nella directory)

git restore --staged <file> | rimuove un file dall'index di git

git status | ottiene i file della repository selezionata con il loro stato (modificato, committato, aggiunto all'index...)

git commit -m "<titolocommit>" -m "<descrizionecommit>" | esegue la procedura di commit, è buona norma inserire titoli e descrizioni esaustive per segnalare meglio i cambiamenti (descrizione non obbligatoria) 

git log | mostra tutti i commit eseguiti

git push -u origin master | manda i file committati al server github

git pull origin | prende i file presenti sul server github
